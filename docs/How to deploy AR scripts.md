# Windows

```
How to install Git on windows :
1. Download the installer from https://git-scm.com/download/win
2. Run the installer and follow the instructions
```

```
How to deploy on windows :
1. Open powershell and type the following command 
    - 'Invoke-WebRequest -Uri https://github.com/fwillems42/ARWazuh/raw/master/deployment/windows/deployment.exe' -o deployment.exe'
2. Run deployment.exe
```

# Linux

```
How to install Git on linux :
1. Download the installer from https://git-scm.com/download/lin or use the package manager of your distro
2. Run the installer and follow the instructions
```

```
How to deploy on linux :
1. Open the terminal and type the following command
    - 'wget https://github.com/fwillems42/ARWazuh/raw/master/deployment/linux/deployment.py
2. Run deployment.py with python
```
